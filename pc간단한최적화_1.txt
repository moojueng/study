CMD 이용한 
간단한 PC 최적화 명령어

1. 명령 프롬프트(CMD) 관리자로 실행

2. ipconfig /flushdns
 - DNS 캐시를 정리하여, 인터넷 속도를 빠르게 해주는 명령어

3. chkdsk /f
 - 디스크의 배드 섹터를 검사하여, 복구하는 명령어
   ( 입력을 하면 "이 볼륨을 검사하도록 하시겠습니까?" 라고 질문할텐데
    y 를 입력해주시고, 재부팅을 하면 적용됩니다! )
   그러면 드라이브가 최적화 되어 pc가 빨라집니다.

4. sfc / scannow
 - 시스템의 무결성 위반 여부를 찾고, 해결해주는 명령어
   ( 완전히 검사가 되기에는 몇 분 시간이 소요됩니다!
    검사가 완료되면 pc가 빨라집니다. )
